# De Plugin Gazette

Wekelijks performantierapport voor pluginbatterij.com

## Live URL
`https://JOUW-GEBRUIKERSNAAM.github.io/plugin-gazette/`

## Structuur
```
plugin-gazette/
├── index.html        ← Meest recente editie (altijd overschrijven)
├── archief.html      ← Overzicht alle edities
├── wk11.html         ← Archief editie week 11
├── wk10.html         ← Archief editie week 10
└── README.md
```

## Elke week updaten
1. Hernoem de huidige `index.html` naar `wkXX.html` (bv. `wk11.html`)
2. Upload de nieuwe Gazette als `index.html`
3. Voeg een link toe in `archief.html`

## GitHub Pages inschakelen
Settings → Pages → Source: Deploy from branch → main → / (root) → Save
